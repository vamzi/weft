# Weft showcase site

Public marketing + ClickBench performance site for Weft, deployed to GitHub Pages at
`https://vamzi.github.io/weft/`. Separate from the control-plane app in `../web` but shares its
design tokens (`src/styles/theme.css`, copied from `web/src/styles/theme.css`).

- **Stack**: React 18 + Vite 5 + Tailwind 3 + TypeScript. `HashRouter` so deep links work on
  Pages with no server config. `base: '/weft/'` (override with `VITE_BASE=/` for a custom domain).
- **Pages**: `/` (overview, ollama-style) and `/performance` (LakeSail-style benchmark charts).
- **Data**: charts read `src/data/benchmarks.json`, regenerated by
  `../bench/clickbench/multi/to-site.py` from the real benchmark run. Engines without a measured
  number render as muted "pending" bars — no figure is ever hand-entered.

```sh
npm install
npm run dev        # http://localhost:5174
npm run build      # → dist/  (what GitHub Actions publishes)
npm run typecheck
```

Deploy is automatic: `.github/workflows/deploy-site.yml` builds and publishes on every push to
`main` that touches `site/**`. One-time repo setup: **Settings → Pages → Source = GitHub Actions**.
